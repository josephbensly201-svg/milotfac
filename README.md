# MilotFac – Site web officiel du préfac communautaire de Milot

![MilotFac](home.jpg)

## 📌 À propos du projet

**MilotFac** est un préfac communautaire à faible coût destiné aux élèves de Milot et des environs qui souhaitent 
préparer les concours d'entrée à l'Université d'État d'Haïti et aux autres universités publiques.

Ce site web est la plateforme officielle du projet. Il permet aux élèves de :

- S'informer sur le préfac (objectifs, règles, contacts)
- S'inscrire en ligne via un formulaire Google Forms intégré
- Recevoir un badge personnel par e-mail après inscription
- Rejoindre le groupe WhatsApp officiel pour échanger avec la communauté

---

## 🛠️ Technologies utilisées

| Technologie        |                Rôle                           |
|--------------------|---- ------------------------------------------|
| HTML5              | Structure du site                             |
| CSS3               | Style et mise en page responsive              |
| JavaScript         | Navigation sans rechargement (style Facebook) |
| Google Forms       | Collecte des inscriptions                     |
| Google Apps Script | Envoi automatique des badges par e-mail       |
| GitHub Pages       | Hébergement gratuit du site                   |
